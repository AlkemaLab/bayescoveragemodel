# Add uncertainty to observations

This function adds uncertainty to observations using the model fit
object.

## Usage

``` r
add_uncertainty_in_obs(fit, perc_low = 0.025, perc_up = 0.975)
```

## Arguments

- fit:

  model_fit object.

- perc_low:

  lower percentile for uncertainty.

- perc_up:

  upper percentile for uncertainty.
