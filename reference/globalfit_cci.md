# Global fit object globalfit_cci

Global fit object globalfit_cci

## Usage

``` r
globalfit_cci
```

## Format

### `globalfit_cci`

list with global fit objects including

- post_summ:

  data frame with posterior summaries for relevant parameters

- geo_unit_index:

  data frame with geo_unit info

- source_index:

  data frame with sources of survey data with indices used in the model

- time_index:

  data frame with TIME info

- other settings:

  spline settings, t_star, etc

- data_from2010only:

  logical indicating to fit to data from 2010 onwards only (TRUE) or
  with all available data (FALSE)
