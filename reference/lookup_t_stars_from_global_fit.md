# Look up t_stars to use in local fit based on the values used in a global fit

Look up t_stars to use in local fit based on the values used in a global
fit

## Usage

``` r
lookup_t_stars_from_global_fit(
  local_geo_unit_index,
  global_geo_unit_index,
  global_t_stars
)
```

## Arguments

- local_geo_unit_index:

  Local geo unit index

- global_geo_unit_index:

  Global geo unit index

- global_t_stars:

  Global t_stars values

## Value

Vector of t_stars for local fit
